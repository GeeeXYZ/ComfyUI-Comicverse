Font Directory
===============

This directory contains font files for the ComicVerse nodes.

Font Files:
-----------
- handwriting.ttf: Handwriting style font (for decorative text)
- black_bold.ttf: Bold sans-serif font (for titles)
- cartoon.ttf: Cartoon/comic style font (for speech bubbles)

Installation:
-------------
1. Download or obtain the font files you want to use
2. Replace the placeholder files with actual .ttf or .otf font files
3. Restart ComfyUI

Font Sources:
-------------
You can download free fonts from:
- Google Fonts (fonts.google.com)
- DaFont (dafont.com)
- Font Squirrel (fontsquirrel.com)

Note: Make sure you have the proper licenses for any fonts you use.
